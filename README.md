## Introduction

CenteralLibrary is a web-based application designed for the University of Sadat City. It provides access to an extensive collection of books, journals, and digital resources, fostering a culture of learning and research.

## Features

- Home page with navigation to different sections
- About Us section detailing the mission and offerings
- Services section showcasing available resources
- Contact Us form for inquiries
- FAQ section for common questions
